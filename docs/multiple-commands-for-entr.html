Use "sh -c" to supply multiple commands to "entr"

December 29, 2025

entr lets you re-run commands when files change. It's great for iterating on code because you don't have to switch contexts; your code just re-runs when you save the file.

If you need to run multiple commands or redirect input/output, just invoke your shell, e.g.:

ls foo.c |entr sh -c "make && make test"
